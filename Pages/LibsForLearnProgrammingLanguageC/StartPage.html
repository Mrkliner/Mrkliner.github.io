Для чего и зачем изучать язык С?

Язык программирования С изучать стоит для понимания логики других языков программирования, в других языках программирования (Java, python, C++, C#, Pascal, JavaScript, PHP, lua) есть такие операторы как if() и тому подобные, они есть и в С.
Язык Оператор Присутствует
С if() Да
Python if() Да
Java if() Да
PHP if() Да
JS if() Да
