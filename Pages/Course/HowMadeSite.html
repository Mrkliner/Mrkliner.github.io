На каких языках программирования пишут сайты?

Сайты пишутся на языках HTML, CSS, JS. HTML - нужен для создание каркаса сайта. CSS - для создание стилей к примеру с помощью CSS можно сменить цвет текста. C помощью JS (Java Script) делается функциональная часть сайта.

HTML выглядит вот так:


<html>
­­­<head>
­­­­­­<title>
<body>
body>
html>
